# Pre-requisites
* Homebrew [https://brew.sh/](https://brew.sh/)
* Node `brew install node` or use NVM [https://github.com/creationix/nvm](https://github.com/creationix/nvm) (recommended)
* Yarn `brew isntall yarn`

# Installation
* `yarn install`

# Development
* `npm run dev`

# Compile
* `npm run build`
